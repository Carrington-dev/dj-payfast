# Adding Bootstrap to Django Admin ## Step 1: Create Template Directory Structure Create the following directory structure in your Django project: ``` your_project/ ├── templates/ │ └── admin/ │ ├── base.html │ ├── base_site.html │ └── change_form.html ``` ## Step 2: Override base.html Create `templates/admin/base.html`: ```html {% load static %} {% block title %}Django Admin{% endblock %} {% block extrastyle %}{% endblock %}
{% block header %}

{% endif %}

{% block nav-sidebar %}
{% block sidebar %} {% endblock %}
{% endblock %}
{% block messages %}
{% if messages %} {% for message in messages %}
{{ message }}
{% endfor %} {% endif %}
{% endblock %}
{% block content %} {% endblock %}
{% block extrajs %}{% endblock %} ``` ## Step 3: Create Custom CSS Create `static/admin/css/custom_admin.css`: ```css /* Custom Bootstrap Django Admin Styles */ /* Tables */ #result_list { @extend .table; margin: 0.25rem 0; } .errorlist li { padding: 0.25rem 0; } ``` ## Step 4: Update settings.py Add the templates directory to your settings: ```python TEMPLATES = [ { 'BACKEND': 'django.template.backends.django.DjangoTemplates', 'DIRS': [BASE_DIR / 'templates'], # Add this 'APP_DIRS': True, 'OPTIONS': { 'context_processors': [ 'django.template.context_processors.debug', 'django.template.context_processors.request', 'django.contrib.auth.context_processors.auth', 'django.contrib.messages.context_processors.messages', ], }, }, ] STATIC_URL = '/static/' STATICFILES_DIRS = [BASE_DIR / 'static'] # Add this if not present ``` ## Step 5: Create change_form.html Create `templates/admin/change_form.html` for Bootstrap-styled forms: ```html {% extends "admin/base_site.html" %} {% load i18n admin_urls static admin_modify %} {% block extrahead %}{{ block.super }} {{ media }} {% endblock %} {% block extrastyle %}{{ block.super }} {% endblock %} {% block content %}
